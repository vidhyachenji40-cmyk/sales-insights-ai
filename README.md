# AI Sales Insights Generator 📈

An automated intelligence system that transforms raw retail sales data into natural-language executive summaries using **Claude Sonnet 4.6**.

## 🚀 Key Features
- **Data Processing:** Uses Pandas to aggregate revenue and identify top-performing products.
- **AI Reasoning:** Leverages Anthropic's latest model to generate strategic business advice based on daily KPIs.
- **Security:** Implements environment variables and `.gitignore` to protect sensitive data and API credentials.

## 🛠️ Tech Stack
- **Python** (Pandas, Anthropic SDK)
- **Model:** Claude Sonnet 4.6
- **Environment:** Google Cloud Shell
